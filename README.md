# LLM API Experiment

This project shows how to interact with a Large Language Model (LLM) using code. 

## What it does
- Sends multiple prompts to an LLM API
- Receives and prints generated responses
- Helps understand how LLMs behave 

## Setup

1. Install dependencies:
   pip install -r requirements.txt
  
2. Set your API key:
   export OPENAI_API_KEY="your_api_key_here"
   
3. Run the script:
  python main.py


## Purpose
This project to learn how to use LLMs through APIs.
